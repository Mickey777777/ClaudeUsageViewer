# ClaudeUsage

Claude API 사용량을 macOS 메뉴바에서 실시간으로 확인하는 앱입니다.

## 기능

- **메뉴바 상시 표시** — 5시간 세션 사용량을 수직 바 + 퍼센트로 표시
  - 초록(0-50%) / 주황(50-80%) / 빨강(80%+)
- **팝업 상세 보기** — 클릭하면 세션(5시간) · 주간(7일) 사용량, 리셋까지 남은 시간 확인
- **자동 갱신** — 1분마다 API 호출, 429 응답 시 3분 후 재시도
- **에러 처리** — 이전 사용량은 유지하고 에러 내용만 하단에 표시

## 요구사항

- macOS 13+
- [Claude Code](https://claude.ai/code) CLI 설치 및 로그인 상태

Claude Code가 Keychain에 저장한 OAuth 토큰을 읽어서 `api.anthropic.com/api/oauth/usage` 엔드포인트를 호출합니다. 별도의 API 키 설정이 필요 없습니다.

> **시스템 암호 입력 창이 뜨는 경우** — 앱이 macOS Keychain에 저장된 Claude Code 로그인 토큰에 접근할 때 시스템이 권한을 요청하는 것입니다. "항상 허용"을 선택하면 이후에는 뜨지 않습니다.

## 설치

1. 이 저장소를 클론
2. Xcode로 `ClaudeUsage.xcodeproj` 열기
3. 빌드 & 실행 (`⌘R`)

## 사용법

| 동작 | 결과 |
|------|------|
| 메뉴바 아이콘 클릭 | 사용량 팝업 열기/닫기 |
| 우클릭 | 새로고침 / 종료 메뉴 |
| `R` 단축키 (우클릭 메뉴) | 즉시 새로고침 |
